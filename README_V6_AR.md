# Ahmed Career OS V6

الجديد:
- بروفايل: صورة، اسم، إيميل، هاتف، مكان، LinkedIn، مسمى مهني.
- حفظ آخر CV داخل التطبيق وفتحه أو تنزيله.
- رسوم إضافية في Dashboard: توزيع الإنجاز وآخر 30 يومًا ومؤشر الاستعداد.
- تسجيل الإجابات بالصوت وحفظها وتشغيلها.
- تحويل الكلام إلى نص عندما يدعمه الجهاز.
- تقييم ذكي محلي مبدئي للإجابة مع نصائح.
- لا يتم إرسال الصورة أو CV أو الصوت إلى الإنترنت.
