# Ahmed Career OS — نسخة أندرويد

هذا مشروع Android كامل يحوّل التطبيق إلى APK حقيقي يعمل دون إنترنت ويحفظ بياناتك على الجهاز.

## استخراج APK من GitHub

1. أنشئ مستودعًا جديدًا على GitHub.
2. ارفع كل الملفات الموجودة داخل هذا المجلد.
3. افتح تبويب **Actions**.
4. اختر **Build Android APK**.
5. اضغط **Run workflow**.
6. بعد اكتمال البناء، افتح نتيجة التشغيل وحمّل الملف:
   **Ahmed-Career-OS-APK**
7. فك الضغط وستجد `app-debug.apk`.
8. انقله إلى هاتفك وثبّته. قد تحتاج السماح بالتثبيت من هذا المصدر.

النسخة Debug مناسبة للاستخدام الشخصي. نشر التطبيق على Google Play يحتاج نسخة Release موقعة بمفتاح خاص.
